Bot de Discord para gestionar Documentos Nacionales de Identidad virtuales

Comenzar Comandos Descargar Docs

✨ Características Principales

🤖 Automático

Genera DNIs automáticamente desde mensajes con formato específico

🎨 Personalizable

3 países y 3 temas de diseño disponibles

💾 Persistente

Base de datos JSON que guarda todos los DNIs

✅ Firma Digital

Sistema de aprobación con firma de administradores

🚀 Instalación Rápida

1. Instalar Python y Dependencias

# Instalar librerías necesarias
pip install discord.py Pillow qrcode[pil]

2. Configurar el Bot

  1. Ve a Discord Developer Portal
  2. Crea una nueva aplicación y bot
  3. Copia el token del bot
  4. Activa los intents necesarios

3. Editar Configuración

ADMIN_ROLE_ID = 1234567890  # ID de tu rol admin
BOT_TOKEN = "TU_TOKEN_AQUI"  # Token del bot

4. Ejecutar el Bot

python index.py
✅ Si ves "Bot conectado como [Nombre]", ¡funciona correctamente!

🤖 Sistema de Solicitudes Automáticas

Los usuarios pueden solicitar DNIs escribiendo un mensaje con este formato:

DNI nuevo
Edad: 18
Fecha de Nacimiento: 15/03/2006
Usuario de Roblox: JuanGamer123
País: españa
Tema: clasico
💡 El bot detecta automáticamente el mensaje y genera el DNI para firma

❓ Preguntas Frecuentes

¿Puedo crear mi propio DNI?

No, solo los administradores pueden crear DNIs. Pero puedes solicitar uno usando el sistema automático en el canal de solicitudes.

¿Los DNIs se pierden si el bot se reinicia?

No, todos los DNIs se guardan en dnis_database.json y persisten entre reinicios.

¿Por qué no recibí mi DNI por DM?

Asegúrate de tener los mensajes privados activados desde miembros del servidor.

¿Cómo sé si mi solicitud fue procesada?

El bot añadirá una reacción ✅ a tu mensaje de solicitud.

⚠️ Importante: Nunca compartas tu token del bot. Es como tu contraseña.
